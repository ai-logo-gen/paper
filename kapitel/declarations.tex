\subsection*{Data Availability}
The dataset used in this study is based on \texttt{iamkaikai/amazing\_logos\_v4} (\url{https://huggingface.co/datasets/iamkaikai/amazing_logos_v4}). The processed dataset, including train/validation/test splits, sketches, and prompts used in this work, is available at Kaggle: \url{https://www.kaggle.com/datasets/paulhornig/minimalistic-logos-sketches-and-prompts}.

\subsection*{Code Availability}
The code for data preprocessing is available at \url{https://github.com/ai-logo-gen/data-prep}. The code for model fine-tuning and evaluation is available at \url{https://github.com/ai-logo-gen/modeling}. The implementation is based on the Hugging Face Diffusers library and uses standard open-source tools including Stable Diffusion v1.5, ControlNet, and LoRA adapters.
